# 동우산삼 홈페이지

GitHub Pages용 정적 홈페이지입니다.

## 공개 방법
1. 이 폴더 안의 모든 파일과 폴더를 GitHub 저장소 최상위에 업로드합니다.
2. 저장소의 Settings > Pages로 이동합니다.
3. Build and deployment에서 Source를 Deploy from a branch로 선택합니다.
4. Branch를 main, 폴더를 /(root)로 설정하고 Save를 누릅니다.
5. 몇 분 뒤 `https://gmlwns70-cmd.github.io/dongwoo-sansam/`에서 확인합니다.
